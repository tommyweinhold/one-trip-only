DATA SOURCES & APIS
OTO runs entirely on free, open APIs — no keys, no accounts, no costs. Ever.
🌤 Weather — Open-Meteo (ECMWF models, free forever)
⛰ Elevation — OpenTopoData SRTM (global 90m resolution)
🌿 Air Quality — OpenAQ (global monitoring stations)
🔍 Place Search — Nominatim / OpenStreetMap
📍 Hotels & POIs — Overpass API (OpenStreetMap data)
🗺 Map Tiles — Esri Topo · Stadia Terrain · OSM Standard · Esri Satellite
📂 GPX Import — Built-in browser parser, works with any GPX file
👥 Community — Supabase (posts/auth) + Cloudinary (photos)
Community features require Supabase + Cloudinary — see setup guide at top of <script>.
